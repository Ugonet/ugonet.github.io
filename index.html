Projets

CampusConnect

Application mobile en React pour les étudiants de l’IUT de Lens (Université d’Artois). Centralise emploi du temps, notes, mails et notifications pour simplifier leur quotidien.

Reborn API

API personnalisée développée via reverse engineering pour accéder au système d’inventaire SumUp du Bureau des étudiants, permettant l’affichage en temps réel des stocks et prix sur écran.

ProNet

Application complémentaire à Pronote qui, via un proxy interne, permet de modifier l’emploi du temps et d’autres infos. Offre aussi la possibilité d’appliquer un thème CSS personnalisé à l’interface.

VoltShift

Extension basée sur un proxy qui modifie les requêtes du site Voltaire, après un reverse engineering, permet de contourner la limitation d'attente de 36h entre deux entraînements.

PSconnect

Application mobile NFC pour un camping permettant de lire les badges douche des campeurs, gérer les jetons et ajouter des crédits, développée après un reverse engineering des bracelets RFID.

ECO²Stat

Projet scolaire mêlant électronique et programmation : des boîtiers relevants la qualité de l’air dans plusieurs zones stratégiques du lycée (cantine, CDI, foyer, extérieur). Les données sont centralisées et affichées sur une carte interactive avec des zones colorées, exportables pour analyses.

WordShaker Booster

Outil qui liste tous les mots valides pris en charge par la grille de Word Shaker, pour optimiser son score et maximiser les points en jeu.
